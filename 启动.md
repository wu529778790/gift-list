# 🚀 快速启动指南

## 1. 启动项目

```bash
npm run dev
```

## 2. 访问系统

打开浏览器：`http://localhost:3000`

## 3. 使用流程

### 第一次使用
1. 自动跳转到**创建事项**页面
2. 填写：
   - 事项名称（例如：张三李四新婚之喜）
   - 开始/结束时间
   - 管理密码（请牢记！）
   - 选择主题（喜事/白事）
3. 点击"创建并进入"

### 录入礼金
1. 在主界面左侧填写：
   - 姓名
   - 金额（自动显示中文大写）
   - 收款类型（现金/微信/支付宝/其他）
   - 备注（可选）
2. 点击"确认录入"

### 查看与导出
- **右侧礼簿**：传统格式展示
- **打印/PDF**：保存纸质版
- **导出Excel**：数据备份
- **开启副屏**：大屏幕展示

## 4. 测试工具

访问：`http://localhost:3000/test-redirect`

可以：
- 检查存储状态
- 清除所有数据
- 测试路由跳转
- 查看实时日志
