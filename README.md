## 📖 使用说明

### 快速开始

**首次使用：**
1. 下载 `index.html`
2. 双击打开
3. 选择：✨ 创建新事件 或 📂 导入备份

**创建新事件：**
1. 填写活动名称、时间
2. 设置密码（默认 `123456`）
3. 选择主题颜色

**导入备份：**
1. 首页点击 📂 导入备份
2. 选择 .json 备份文件
3. 智能合并，自动跳过重复记录

### 数据管理

**导出备份：**
1. 进入主界面
2. 点击 💾 导出备份
3. 保存 .json 文件

**导入备份：**
1. 首页或主界面点击 📂 导入备份
2. 选择备份文件
3. 导入成功显示统计信息

**多设备同步：**
1. 设备A：导出备份 → 发送文件（微信/QQ/邮件）
2. 设备B：导入备份 → 自动加载所有数据

### 日常使用

**录入礼金：**
1. 点击"录入礼金"
2. 输入姓名和金额
3. 自动显示中文大写金额
4. 保存即可

**查看统计：**
- 自动计算总金额
- 显示人数统计
- 支持搜索和筛选

**其他功能：**
- **打印**：生成精美纸质版
- **Excel**：导出表格存档
- **副屏**：现场大屏展示

## 💡 小贴士

- 🔑 **记住密码**：忘记密码无法找回数据

- 🔄 **多设备同步**：导出 → 发送 → 导入，轻松同步

## ❓ 常见问题

**Q: 需要安装软件吗？**
A: 不需要！只有一个 HTML 文件，双击就能用。

**Q: 数据安全吗？**
A: 非常安全。数据加密后存储在本地浏览器，不会上传到任何服务器。

**Q: 数据会丢失吗？**
A: 浏览器清除数据会丢失。务必使用"导出备份"功能定期备份！

**Q: 如何在多设备间同步？**
A: 设备A导出备份 → 发送文件（微信/QQ/邮件）→ 设备B导入备份。

**Q: 备份文件安全吗？**
A: 非常安全。备份文件包含加密数据，没有密码无法查看。

**Q: 导入时会重复数据吗？**
A: 不会。系统会智能识别重复记录，自动跳过。

**Q: 会收费吗？**
A: 永久免费，开源项目。

**Q: 支持哪些浏览器？**
A: Chrome、Firefox、Edge、Safari 等现代浏览器都支持。
