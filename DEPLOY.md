# 🚀 快速部署指南

## 📦 获取应用

```bash
# 1. 访问 GitHub Releases 页面
# 2. 下载 index.html
# 3. 双击即可使用！
```

就这么简单，不需要其他步骤。

---

## 🎯 使用方式

### 直接打开（✨ 推荐）
- 双击 `index.html`
- 或在浏览器按 `Ctrl+O` 选择文件
- **无需任何服务器！**

### 部署到服务器
上传 `index.html` 到 Web 服务器即可。

### 部署到 GitHub Pages
```bash
git init && git add index.html && git commit -m "Deploy"
git remote add origin <your-repo>
git push -f origin main:gh-pages
```

- **密码**: `123456`

## 📤 数据备份与恢复

### 导出备份
在主界面点击 **💾 导出备份** 按钮，下载 JSON 备份文件。

### 导入备份
在主界面点击 **📂 导入备份** 按钮，选择备份文件恢复数据。

### 多设备同步
1. 在设备A导出备份
2. 发送备份文件到设备B（微信、QQ、邮件等）
3. 在设备B导入备份文件

---

## ⚠️ 注意事项

1. **数据安全**: 所有数据在本地加密存储，忘记密码无法找回
2. **定期备份**: 务必定期导出备份，防止浏览器数据丢失
3. **浏览器兼容**: 支持现代浏览器（Chrome, Firefox, Edge, Safari）
4. **隐私**: 数据仅存储在本地浏览器，不会上传到服务器

---

## 🆘 常见问题

**Q: 双击打开后显示空白？**
A: 确保使用现代浏览器，或尝试用浏览器打开（Ctrl+O）

**Q: 如何打开副屏？**
A: 在主界面点击"开启副屏"按钮

**Q: 数据会丢失吗？**
A: 清除浏览器数据会丢失。务必使用"导出备份"功能定期备份！

**Q: 如何在多设备间同步？**
A: 导出备份 → 发送到其他设备 → 导入备份
