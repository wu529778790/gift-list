# 📝 输入框样式统一说明

## ✅ 已完成的优化

所有输入框现在使用**统一的样式系统**，确保一致的外观和交互体验。

---

## 🎨 输入框样式特性

### 1. 基础样式 (所有输入框)
```css
/* 适用元素 */
input[type="text"],
input[type="number"],
input[type="date"],
input[type="time"],
input[type="password"],
input[type="email"],
textarea,
select
```

**样式特征**:
- ✅ 内边距: `p-3` (12px)
- ✅ 圆角: `rounded-lg` (8px)
- ✅ 边框: 1px 灰色边框
- ✅ 背景: 白色
- ✅ 宽度: 100%

### 2. 交互状态

#### 默认状态
```
┌─────────────────────────────┐
│ 事项名称 (例如: ...)        │
└─────────────────────────────┘
```
- 边框颜色: 主题边框色
- 背景: 白色

#### 悬停状态 (Hover)
```
┌─────────────────────────────┐
│ 事项名称 (例如: ...)        │ ← 鼠标悬停
└─────────────────────────────┘
```
- 边框颜色: 主题主色
- 阴影: 轻微阴影

#### 焦点状态 (Focus)
```
┌─────────────────────────────┐
│ 事项名称 (例如: ...)        │ ← 输入框获得焦点
└─────────────────────────────┘
```
- 边框颜色: 主题主色
- 环形焦点: 主题环颜色
- 阴影: 中等阴影

#### 禁用状态 (Disabled)
```
┌─────────────────────────────┐
│ 事项名称 (例如: ...)        │ ← 灰色，不可编辑
└─────────────────────────────┘
```
- 背景: 浅灰色
- 透明度: 60%
- 光标: 禁止符号

## 🎯 主题系统

### 喜庆红主题 (喜事)
```css
/* 焦点环颜色 */
--primary-ring-color: #ef4444;

/* 边框颜色 */
--primary-border-color: #ffaaaa;

/* 主色 */
--primary-color: #c00;
```

**输入框效果**:
- 默认: 浅红边框
- 悬停: 深红边框
- 焦点: 红色环 + 深红边框

### 肃穆灰主题 (白事)
```css
/* 焦点环颜色 */
--primary-ring-color: #4b5563;

/* 边框颜色 */
--primary-border-color: #d1d5db;

/* 主色 */
--primary-color: #374151;
```

**输入框效果**:
- 默认: 灰色边框
- 悬停: 深灰边框
- 焦点: 灰色环 + 深灰边框

```tsx
  <option value="festive">喜庆红 (喜事)</option>

```

```tsx

```

## 🔧 技术实现

### CSS 优先级
```css
/* 1. 基础样式 (最低优先级) */
input { ... }

/* 2. 悬停状态 */
input:hover { ... }

/* 3. 焦点状态 */
input:focus { ... }

/* 4. 主题增强 (最高优先级) */
.themed-ring:focus { ... }
```

### Tailwind + CSS 变量
```css
/* 使用 CSS 变量实现主题切换 */
border-color: var(--primary-border-color, #d1d5db);
--tw-ring-color: var(--primary-ring-color, #4b5563);
```

```css
/* 自定义单选框颜色 */
input[type="radio"].themed-ring {
  accent-color: var(--primary-color);
}
```

### 3. 响应式
```css
/* 移动端优化 */
@media (max-width: 768px) {
  input, textarea, select {
    padding: 0.75rem;  /* p-3 对应 12px，移动端保持一致 */
  }
}
```
